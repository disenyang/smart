# smart
smart.js 前端mvc轻量级框架，原生方式的双向数据绑定，支持表达式，for，if等形式，支持语义标签式开发。立志于前端轻量级mvc开发平台，适合网站开发，微信h5开发，后台管理。根据数据模型一键生成常用代码，生成的代码是标准格式的代码，统一代码风格，让开发只关注于业务逻辑，摒弃之前新增模块试的复制黏贴功能。

                <td>${row.sex==0?"男":"女"}</td>
            </tr>
        </table>
